# Chat Generator

This is a yeoman generator of chat.
With color and language option.

## Getting Started

```bash
$ npm install -g yo
```

```bash
$ npm install -g generator-chat
```

Finally, initiate the generator:

```bash
$ yo chat
```

###Developpement

To launch in developpement:

```bash
$ npm install
$ node server.js
```

To launch in release:

```bash
$ npm install
$ grunt
$ cd public/
$ node server.js
```

NOTE: public/ is the dist folder.
